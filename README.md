# Choose lunch App

## このアプリの目的

みなさんは在宅勤務の日々の中で
今日のお昼は何作ろうかな〜と考えないといけない時間が意外と多いのではないでしょうか？

そんな時にこのアプリを使えば、
厳選された、短時間でパパっと作れる主食レシピの中から
いくつか選んで提案してくれます！

主食のジャンルを選べば、絞り込んで提案が可能です。

なんでもいいから決めてほしい！という時は
ぜひランダムボタンを押してみて！
すべての登録されたレシピから 1 つを選んで提案してくれることでしょう！
